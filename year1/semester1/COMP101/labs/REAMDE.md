# COMP101 Labs – Semester 1

This folder contains lab exercises for **COMP101** focused on basic Python programming concepts.

## Files in this folder

- `Lab-01_sequence.py`  
  A working example of **sequential code**.  
  It demonstrates how Python executes statements one-by-one in order.

- `Lab-01_sequence_bugged.py`
  A bugged version of the same exercise, intended for debugging practice.  
  It includes syntax and naming errors to identify and fix.


---

## Learning objectives

By completing these labs, you should be able to:

- Understand what **sequential execution** means.
- Use `print()` and `input()` for simple console interaction.
- Recognize common Python syntax mistakes.
- Debug beginner-level Python programs.

---

## How to run

1. Open a terminal in this folder.
2. Run a file with Python 3:

```bash
python3 Lab-01_sequence.py
```


> Depending on your system, you may use `python` instead of `python3`.
